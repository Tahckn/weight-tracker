# Weight Tracker

Weight Tracker app. 

Created with Vue.js, chart.js and Tauri for Windows app.

 [Website version](https://relyonrhythm.netlify.app/).

## Installation
```bash
Export the .msi file from zip and install thats it.
```

## Usage
 ```bash
npm run install

npm run tauri build - (for build windows app)
```
